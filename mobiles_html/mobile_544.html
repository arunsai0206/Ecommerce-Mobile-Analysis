    • Description
    The itel 2175Pro features an innovative design with a 2-inch display and a premium camera deco. It offers wireless FM recording and vibration support, ensuring an enriched user experience. The phone is equipped with the King Voice function, facilitating easy navigation through voice commands. With expandable memory support of up to 32GB and Dual SIM functionality, it provides ample storage and connectivity options. The 1200mAh battery ensures worry-free battery backup. Additionally, it boasts language support for nine languages, including English, Hindi, Tamil, Telugu, Gujarati, Bengali, Malayalam, Punjabi, and Kannada.
